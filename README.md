# 📊 Securitisation Risk Analytics Dashboard

A comprehensive Power BI analytics solution designed for banking and financial institutions to monitor securitisation portfolios, assess credit risk, and support executive decision-making.

---

## Project Overview

This project demonstrates an end-to-end Power BI solution built using banking loan portfolio data. The dashboards provide insights into portfolio quality, delinquency trends, IFRS 9 Expected Credit Loss (ECL), vintage performance, and stress testing scenarios.

The solution is designed to simulate an enterprise-level risk analytics environment commonly used by banks, NBFCs, and financial institutions.

---

## Objectives

- Monitor portfolio health
- Assess securitisation risk
- Analyze loan delinquency
- Estimate Expected Credit Loss (ECL)
- Evaluate vintage performance
- Perform stress testing using dynamic risk scenarios
- Support executive-level decision making

---

# Dashboard Modules

## 1. Executive Risk Overview

Provides a high-level summary of portfolio performance including:

- Total Pool Balance
- Total Loans
- Average Interest Rate
- Average CIBIL Score
- Expected Credit Loss
- Probability of Default
- Loss Given Default
- Portfolio Exposure
- Vehicle-wise Exposure
- State-wise Risk Distribution

---

## 2. DPD Analytics

Analyzes loan delinquency using:

- DPD Bucket Distribution
- DPD Exposure
- SMA Classification
- High Risk Loan Table
- Roll Rate Migration
- NPA Ratio
- Delinquency KPIs

---

## 3. IFRS 9 ECL Analytics

Supports Expected Credit Loss reporting including:

- Stage 1 ECL
- Stage 2 ECL
- Stage 3 ECL
- Total ECL
- IFRS 9 Stage Mix
- Probability of Default Distribution
- Loss Given Default Distribution
- State-wise Expected Credit Loss
- Vehicle-wise Expected Credit Loss

---

## 4. Vintage Analysis

Tracks portfolio performance over time using:

- Vintage Performance Curve
- Pool Factor Runoff Curve
- Gross Loss Trend
- Net Loss Trend
- Recovery Trend
- Vintage Performance Table

---

## 5. Stress Testing & Scenario Analysis

Interactive scenario analysis including:

- PD Stress Parameter
- LGD Stress Parameter
- Stressed Expected Credit Loss
- Incremental Loss
- Stress Impact Percentage
- State-wise Stress Impact
- Vehicle-wise Stress Impact
- Portfolio Stress Analysis
